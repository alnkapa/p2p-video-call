## 🎯 Usage

### Creating a call:
1. Click "Create Call"
2. Allow camera and microphone access
3. Share link or QR code

### Joining a call:
1. Follow the received link
2. Or enter session ID manually
3. Allow media device access
4. Start communicating!

### Call controls:
- **📹** - Toggle video
- **🎤** - Toggle microphone
- **📷** - Switch camera (on mobile)
- **📞** - End call
